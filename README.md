# 👨‍💼 Employee App (Flutter)

A simple Employee Management App built with **Flutter**, using **Clean Architecture** and **BLoC state management**, integrated with `reqres.in` API.

---

##  Features

### Authentication
- Login with email (magic code flow)
- Verify token login
- Persistent session using local storage
- Auto login if token exists

### Employee Management
- List employees
- Search employees (name / phone)
- View employee details
- Add new employee
- Edit employee
- Delete employee
- Toggle favorite employee

### Tech Stack
- Flutter
- BLoC (flutter_bloc)
- Dio (HTTP client)
- SharedPreferences (local storage)
- Clean Architecture
- Dependency Injection (GetIt)


### Screens
- Login Screen
- Employee List Screen
- Employee Detail Screen
- Add / Edit Employee Screen
